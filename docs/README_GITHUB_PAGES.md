# Mindful Bump Free Hosting Notes

This `docs/` folder is ready for free static hosting with GitHub Pages.

## Files

- `index.html`
- `privacy.html`
- `support.html`

## GitHub Pages

1. Put this `docs/` folder into a public GitHub repository.
2. Open the repository settings on GitHub.
3. Go to `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose your default branch.
6. Choose the `/docs` folder.
7. Save.
8. Wait for GitHub Pages to publish.

Your URLs will look like:

- `https://<your-username>.github.io/<repo-name>/privacy.html`
- `https://<your-username>.github.io/<repo-name>/support.html`

## App Store Connect

Use the published public URLs for:

- `Privacy Policy URL`
- `Support URL`
