Mindful Bump

Mindful Bump is a Screen Time based mindfulness buffer app that helps you pause before opening selected apps or categories.

Mindful Bump 是一款基于 Screen Time 的正念缓冲应用，帮助你在打开指定 App 或类别前先停顿一下。
